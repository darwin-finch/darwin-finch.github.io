01

INSTALL & RUN

One curl command installs the binary. The REPL starts in under 100ms. On first run, Finch downloads a Qwen model in the background â€” you can start asking questions immediately.

02

QUERY LOCALLY

All queries run on your local Qwen 2.5 model. Apple Silicon gets Metal acceleration via CoreML. Linux x86_64 uses CPU. No internet required after the model is cached.

03

GIVE FEEDBACK

Press Ctrl+G (good) or Ctrl+B (bad) on any response. Mark truly critical feedback as high-weight (10Ã—). Medium feedback is 3Ã—. Normal is 1Ã—.

04

MODEL ADAPTS

In the background, Finch fine-tunes LoRA adapters on your feedback. The model gradually specializes to your codebase, patterns, and preferences. Adapters are ~5MB each.
